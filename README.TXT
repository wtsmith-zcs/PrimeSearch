PROJECT TITLE: PrimeSearch
PURPOSE OF PROJECT: Allow student to focus on programming a binary search
VERSION or DATE:
HOW TO START THIS PROJECT:  Download prime files of project page
AUTHORS:WT Smith
USER INSTRUCTIONS:

Be sure to download premade prime files. Otherwise you can generate a primeFile with the
main method in CreatePrimeFile.  

CreatePrimeFile main()
The first argument is the number of coins to create.  
This will automatically create a file with the name "CoinsUpTo<maxSize>.txt".
The optional 2nd argument is the name of the file.

PrimeFile
This class will read a primeFile.  The prime file begins with the NUMBER (not max size)
of primes in the file and then has a list of prime numbers.

PrimeSearchRunner
This is already setup to run.  You must code the search routine.
